# План реализации Rezon

> Источник требований: [BRIEF.md](BRIEF.md). Этот файл — дорожная карта: что, в каком порядке, что блокирует, что нужно от владельца.
> Даты: тёплый запуск — 1 августа 2026, публичный — 15 августа 2026.

## Где мы сейчас (этап 0)
- **Прототип**: `index.html` (~1022 строки, vanilla JS, состояние в localStorage). 5 категорий, 14 упражнений, дыхательные и речевые тренажёры, AI-коуч через `/api/chat.js`, прогресс/ачивки, заглушка подписки. Палитра — indigo (меняем).
- **Бот-воронка** (`bot.py`, Railway) — готов: приветствие, рассылка, `/test`.
- **Деплой** на Vercel настроен (`vercel.json`).

## Главная архитектурная развилка (понять заранее)
Сейчас все данные живут в браузере (localStorage): привязаны к устройству, не переживают переустановку, и **нет надёжного способа проверять оплату**. Для платной подписки и сохранения прогресса между устройствами нужен бэкенд: сервер проверяет Telegram-пользователя (`initData`) → база хранит статус подписки, план, прогресс, голосовой дневник.

Поэтому проект делится на 4 слоя, и порядок важен.

|---|---|---|
| **1. Дизайн-система + ребрендинг** | Палитра «Тёплая студия», шрифты Montserrat/Inter, переименование в Rezon | нет |
| **2. Экран «Сегодня» + цикл** | Один активный шаг → Объяснение → запись ДО → упражнение → запись ПОСЛЕ → Финиш | нет (видео — заглушки) |

## Что нужно от владельца (по мере этапов)
- **Сейчас:** ничего — слой 1 можно начинать.
- **Решение:** точный тон/текст системного промпта AI-коуча (предложу черновик на этапе 3).
- **Перед слоем 2:** согласие на Supabase (помогу завести пошагово).
- **Перед слоем 3:** аккаунт Lava.top + API-ключи + продукты/цены в Lava.
- **Контент:** видео/аудио/тексты упражнений — для финального наполнения.
- **Каждый деплой:** свежий GitHub-токен на пару минут (старый удалён).

## Технические заметки и риски
- **Запись голоса в Telegram Mini App:** `MediaRecorder` работает по HTTPS (есть) + разрешение микрофона; на iOS есть нюансы — заложу фолбэки.
- **Видео-аватары:** mp4 из HeyGen, хостинг на Vercel/CDN; сейчас — заглушки.
- **Anthropic API:** `/api/chat.js` уже есть; для плана добавлю отдельный промпт. Токены = расход на Anthropic-аккаунте — держим промпты компактными.
- **Стек:** остаёмся на «ванильном» JS без сборщиков (простой Vercel-деплой). При росте фич аккуратно разобью `index.html` на модули.
