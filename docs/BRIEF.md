## 0. Контекст проекта
**Название приложения: Rezon.** Telegram Mini App для тренировки голоса, речи и уверенности. Целевая аудитория — русскоязычный СНГ-рынок. Бизнес-модель — freemium-подписка, низкий чек, массовый охват (модель Duolingo, не премиум-нишевый продукт).
Бренд курса и воронки в боте остаётся «Голос на миллион» (личный бренд Алмата) — само приложение называется Rezon. Используй название Rezon во всех текстах интерфейса, заголовках, экране приветствия и т.д.
**Уже существует:**
- GitHub: `github.com/sultumovv-lab/golos-app`
- Деплой: `golos-app.vercel.app` (Vercel)
- Бот: `@kaifgolos_bot` (Railway) — приветствие, админ-рассылка, команда `/test` для владельца
- AI-коуч подключён через Anthropic API (`/api/chat.js`)
- Базовый прототип с 14 упражнениями в 5 категориях (дыхание, дикция, резонанс, темп, уверенность)
**Цель этого ТЗ** — переработать приложение под финализированную логику ниже. Дата запуска для тёплой аудитории — 1 августа 2026, публичный запуск — 15 августа 2026.

## 1. Дизайн-система — «Тёплая студия»
### Принцип позиционирования
НЕ премиальный/холодный (противоречит freemium и массовому охвату). НЕ generic-SaaS техно-стиль (indigo и подобное — нет связи с продуктом). Тёплые, энергичные, насыщенные тона — ассоциация с живым человеческим голосом, смелостью говорить, действием.
### Палитра
```
Фон (основной):        #1C1A22
Фон (нав-бар/тёмный):   #161419
Поверхность/карточка:   #28252F
Акцент (основной):      градиент #FF7A5C → #E85D43 → #C9492F
Акцент (вторичный):     #FFB199 (мягкий персиковый)
Успех/прогресс:         #4ADE80
Текст основной:         #FFFFFF
Текст приглушённый:     rgba(255,255,255,0.5–0.7)
```
### Категорически НЕ использовать
- Мультяшных персонажей/иллюстрированных человечков (как у Duolingo) — это размывает главное УТП продукта
- Indigo/фиолетовую SaaS-палитру
- Слоновую кость/золото (премиальная эстетика противоречит freemium)
### Что использовать вместо иллюстраций
**Единственный «персонаж» приложения — реальный человек (Алмат), через AI-аватар (HeyGen) и клонированный голос (ElevenLabs).** Один и тот же аватар — голос приложения для всех пользователей (как у Vocal Image/BoldVoice, но с реальной узнаваемостью и доверием аудитории). Абстрактные звуковые волны — допустимый поддерживающий визуальный элемент, не замена аватару.
### Типографика
- Заголовки: Montserrat / Manrope
- Текст: Inter
- Возможен акцентный serif (Georgia) для отдельных editorial-цитат (например «совет дня»)

## 2. Полная логика экранов
### 2.1 Онбординг (первый запуск)
```
Открытие (без регистрации)
    ↓
Тест — 5 вопросов, ~60 секунд
    ↓
Персональный план от AI (определяет фокус + сроки результата)
    ↓
3 дня бесплатного полного доступа к плану
    ↓
Paywall → Оплата
```
**Вопросы онбординг-теста (финальная версия):**
1. Какая главная проблема с голосом? (тихий / монотонный / дрожит от волнения / слишком быстро / нечёткая дикция)
2. Когда голос подводит чаще всего? (переговоры / публичные выступления / личные разговоры / видео и подкасты)
3. Есть ли скоро важное событие, к которому готовишься? (собеседование / выступление / переговоры / нет конкретного повода) — используется для срочности и персонализации текста на paywall
4. Уже пробовал заниматься голосом раньше? (новичок, ничего не пробовал / было немного опыта / занимался регулярно) — влияет на сложность первых упражнений
5. Сколько времени готов тратить в день? (5 мин / 10-15 мин / 20+ мин)
6. Финальный шаг — голосовая запись: «Прочитай эту фразу вслух, 10 секунд» — даёт реальный аудио-образец для персонализации плана и стартовую точку для дальнейшего сравнения прогресса (механика «до/после»)
Вопрос про абстрактную «главную цель» сознательно убран — пересекался по смыслу с Q1, а срочность (Q3) и контекст (Q2) дают более конкретный материал для персонализации чем общая формулировка цели.
**Важно:** оплата размещена ПОСЛЕ того как человек уже получил персональный план и попробовал его — момент максимальной готовности платить, не до.
### 2.2 Ежедневный цикл (главный паттерн всего приложения)
**Ключевой принцип: ноль решений у пользователя.** Человек никогда не выбирает что делать — приложение уже решило за него.
```
Главный экран «Сегодня» — виден ТОЛЬКО один активный шаг
    ↓
Экран «Объяснение» — видео аватара: суть упражнения + ожидаемый результат + срок
    ↓
Запись голоса ДО упражнения
    ↓
Экран упражнения — полноэкранное видео аватара, таймер, «смотри и повторяй»
    ↓
Запись голоса ПОСЛЕ упражнения
    ↓
Экран «Финиш» — награда, прогресс, XP
    ↓
↻ возврат на «Сегодня», следующий шаг из очереди становится активным
```
**Главный экран «Сегодня» содержит:**
- Карточка-приветствие с местом под фото/видео аватара (не маленькая иконка — заметное присутствие)
- Недельный трекер прогресса (точки Пн–Вс, как у конкурентов)
- Индикатор «энергии»/количества доступных упражнений в день (создаёт ценность, ограничивает злоупотребление в бесплатном тарифе)
- Карточка активного шага (крупная, акцентный цвет)
- Очередь следующих шагов (приглушённая, без возможности кликнуть вперёд порядка)
### 2.3 Прочие табы (не обязательны, не прерывают главный путь)
| Таб | Роль | Логика |
|---|---|---|
| **Сегодня** | Главный, обязательный путь | Один активный шаг за раз |
| **AI-Коуч** | Опционален | Свободный чат, не часть плана |
| **Коллекция** | Опционален | Все упражнения вне плана, бонус для тех кто хочет больше |
| **Профиль** | Пассивный | Голосовой дневник (до/после со временем), достижения, статистика — только наблюдение, не действия |

## 3. Freemium-модель
**Бесплатно навсегда:** 1 упражнение из каждой категории (5 категорий = 5 упражнений)
**Бесплатно временно:** полный персональный план открыт 3 дня сразу после онбординг-теста (момент максимальной мотивации)
**Платно ($7/мес):** всё остальное — полный каталог, AI-коуч без ограничений, геймификация, голосовой дневник
**Для тёплой аудитории (запуск 1 августа):** специальная цена $4-5/мес первые 48 часов, та же воронка (тест → план → оплата), без отдельного пути

## 4. AI-коуч — техническое задание на «личность»
Системный промпт коуча НЕ должен быть общим «ты эксперт по голосу». Нужна явно прописанная методологическая база:
**Методология:**
- Техники из актёрского мастерства / школы сценической речи (работа с диафрагмой, резонаторами, артикуляцией)
- Принципы ораторского искусства (структура убедительной речи, паузы, темп)
- Психология уверенности (работа со страхом, зажимами, тревогой)
**Логика построения плана:**
1. Получает ответы онбординг-теста
2. Определяет 1–2 главных фокуса (не размывает на всё сразу)
3. Подбирает упражнения из каталога именно под эти фокусы
4. Выстраивает прогрессию (база → сложное)
5. Каждый день обновляет очередь на основе того что уже пройдено
**Оценка сроков результата (честная, не произвольная):**
- База/привычка — 2–3 недели
- Заметные изменения — 6–8 недель
- Устойчивый навык — 3–4 месяца
Корректируется в зависимости от серьёзности проблемы и частоты тренировок выбранной пользователем.
**Отдельная функция чата (таб AI-Коуч):** свободные вопросы, ответы 2-4 абзаца, тёплый и конкретный тон, без воды.

## 5. Платёжная интеграция — Lava.top
- API: `https://gate.lava.top/docs`
- Нужно два вебхука: «результат платежа» (первый платёж) + «регулярный платёж» (последующие списания)
- Подписка настраивается по периодам (1/3/6/12 месяцев), поддержка нескольких валют
- Технически: Mini App делает запрос к Lava API → получает ссылку на оплату → открывает её → после оплаты Lava шлёт вебхук на наш сервер → сервер открывает пользователю полный доступ
- Если списание не проходит — Lava делает повторные попытки через 8 и 24 часа перед отменой подписки (нужно учитывать в логике доступа — не закрывать мгновенно при первой неудаче)

## 6. Контент-каталог
**Полная структура (6 месяцев, 60 упражнений, 4 уровня)** — см. master-plan документ для деталей по уровням.
**Минимум для запуска 15 августа:**
- 20 упражнений (уровни 1–2)
- 5 вводных уроков (теория, живое видео, не аватар)
- Каждое упражнение: видео аватара для экрана «Объяснение» (10-15 сек) + видео для самого упражнения (2-4 мин) + текст ожидаемого результата
**Производство контента:**
- Голос: ElevenLabs, клонирование с 10-15 мин записи
- Видео-аватар: HeyGen, на основе 2-3 мин исходного видео

## 7. Геймификация
- Дневной/недельный стрик (трекер точками Пн–Вс)
- «Энергия» — ограничение количества упражнений в день (мотивирует ценить каждое, ограничивает злоупотребление free-tier)
- XP и уровни за каждое упражнение
- Достижения (как в текущем прототипе — расширить набор)
- Голосовой дневник — хранение записей до/после по времени, видно в Профиле как график/сравнение
**Без:** социального соревнования («арена»), детальной ML-аналитики голоса с точными процентами — это следующие этапы (Q4 2026+), не для запуска.
