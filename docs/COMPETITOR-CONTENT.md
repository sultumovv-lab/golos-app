# Разбор контента конкурентов (скрины + видео от владельца, июнь 2026)

> Материал прислал владелец: скрины Vocal Image (Коллекция + Дневной план) и видео ролевой игры Yoodli. Ключевое решение: **аватар ИИ оживлять НЕ нужно — статичное фото персонажа достаточно** (Yoodli сам так делает).

## Yoodli — ролевая игра (из видео)
Формат: **видеозвонок-сетка как в Zoom**, несколько AI-персонажей одновременно, **каждый = статичное фото + имя + должность** (НЕ анимирован). Пользователь — отдельная плитка со своей камерой.
- Пример сценария: продажа совету TechTrust. Персонажи: Thomas Reynauld (CFO), Leslie Thorne (Head of Product), Neil Peterson (Procurement Lead) — у каждого свой интерес/возражения.
- Внизу — активный спикер + субтитры его реплики. Управление: скорость (1x), субтитры (CC), пропуск (→), пауза, стоп.
- **Суть фичи (мульти-персонаж):** имитация совещания/комитета, где надо убедить нескольких людей с разными мотивами. Это флагман Yoodli — и это просто фото + голос + сетка.

## Vocal Image — структура (из скринов)
**Нав:** Сегодня · Коллекция · Арена · Профиль.

**«Дневной план»** = вертикальный путь (таймлайн) из шагов РАЗНЫХ типов:
- 🎭 **Ролевая Игра** (3–5м) — напр. «Реакция на пассивную агрессию»
- 🏋️ **Тренировка** (3–5м)
- 📻 **Радио** (1м) — короткий аудио-урок/совет (напр. «Сила приятности»)
- 🗣️ **Скороговорка** (1м) — напр. «Кукушка кукушонку»

**Валюты/удержание:** 🔥 стрик + 💧 (заморозка) + ⚡ энергия (0/20). Недельный трекер с ✗ за пропущенные дни (визуализация потери).

**«Коллекция»** = библиотека по крупным темам, у каждой % прохождения + подкатегории:
- **Ролевые игры:** Карьера, Общение, Конфликт, Воспитание
- **Коммуникация:** Публичные выступления, Интонация, Чёткость речи, Уверенность
- **Контроль голоса:** Маскулинизация, Феминизация, Восстановление, Сексуальный голос, Разминки
- **Английский акцент:** Американский, Британский
- **Язык тела**

**Арена** = соц/соревновательный слой (голосовые баттлы).

## Выводы для Rezon
1. **Ролевые игры — статичные фото персонажей** (как Yoodli). Добавить **мульти-персонажный режим** («совещание»: 2–4 персонажа с разными мотивами) — это флагман Yoodli, а у нас почти бесплатно на Claude.
2. **Шире таксономия ролевых игр** под жизнь: Карьера / Общение / Конфликт / Воспитание / Свидания (а не только «голос технически»). «Реакция на пассивную агрессию» — отличный бытовой сценарий.
3. **Разнообразие типов шагов** в дневном плане: техника + роль-плей + **мини-урок «Радио»** (1-мин аудио от Алмата) + скороговорка. Вариативность держит.
4. **Лучше, чем у них:** наши персонажи адаптивные (живой Claude), единый разбор от Алмата, русский язык/СНГ-контекст, реальный Алмат (доверие), Telegram-уведомления, AI-персонализация плана, низкая цена.
5. **НЕ копировать сразу:** ниши голоса (маскулинизация/феминизация/акцент) — размывают массовое позиционирование. Арена — после бэкенда и аудитории.
